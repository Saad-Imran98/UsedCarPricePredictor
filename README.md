## How to run (Docker)
### Software Requirements
- Docker
- Docker compose

### Steps
1. Clone this project
2. cd into src/rest_api
3. RUN `sudo docker compose up`
4. Browse to `http://localhost:8000`

## How to run (Manual)
### Software Requirements
- node
- npm
- python
- virtualenv

### Steps
1. Clone this project
2. cd into src/rest_api
3. RUN `chmod +x install.sh setup.sh start.sh `
4. RUN `./install.sh`
5. RUN `./start.sh`
6. Browse to `http://localhost:8000`
