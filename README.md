## How to run (Docker)
### Software Requirements
- Docker
- Docker compose

### Steps
1. Clone this project
2. RUN `sudo docker compose up`
3. Browse to `http://localhost:8000`

## How to run (Manual)
### Software Requirements
- node
- npm
- python
- virtualenv

### Steps
1. Clone this project
2. RUN `chmod +x install.sh setup.sh start.sh `
3. RUN `./install.sh`
4. RUN `./start.sh`
5. Browse to `http://localhost:8000`
